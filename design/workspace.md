# Workspaces

This is a preliminary design document with minimal content.

## Organization Tool

A user may want to use workspaces as an organization tool. The idea is that a project might be split into multiple sessions or work targets. The sessions might be looking at the same target in different ways and need to be grouped together into a project, as opposed to a different project for each session. So instead of creating new projects, multiple sessions are kept in one project and workspaces may be used to represent each of those sessions.
